Wine Quality Classification

This project applies machine learning models to classify wine quality using a cost-sensitive approach.

Key Insight

Random Forest achieved the highest profit by minimizing costly false positives.

Business Impact

The model with the best traditional metrics was not the most valuable one. This project shows the importance of aligning model evaluation with business objectives.

Results

  • Logistic Regression: 563
  • Random Forest: 1713
  • XGBoost: 1468
